// FEATURE GUIDE
MAIN HUD
Tap your player name to edit it inline — max 12 characters
Tap SET YOUR STATS badge to see the band reference table
RECALL LAST loads your most recent logged reading into the sliders
USE AS-IS confirms all sliders at their current position — no movement required
Tap the lock icon 🔓 on any stat to hold it — reset and recall won't change it
Add an optional note when logging to capture context behind the numbers
DYSREGULATED condition fires when your stats show high variance — body and mind out of sync
GRAPH & HISTORY
Tap GRAPH in the footer to view your reading history with trend lines per stat
Use date range filters (7D / 30D / 3M / 6M / 12M / ALL) to focus the graph
Log a reading for a past date using the date picker at the bottom of the graph overlay
Tap EDIT on any logged reading to reload its values into the sliders
Tap DEL to permanently delete a reading
Past notes are shown under each reading in the history list
SETTINGS
Set your personal baseline in Profile — condition thresholds shift relative to your centre, not a fixed scale
Add a note to self in Profile — appears on the main HUD below your condition description
Set your timezone in Profile — clock display updates to match
Tap the ◉ percentage next to the text size slider to type a value directly (80–140)
Tap the version number at the bottom of settings to view the full changelog
Set a Google Calendar reminder for your daily check-in via the reminder button
Colourblind mode switches all stat colours to a high-contrast accessible palette
EXPORT / RESTORE
Use BACKUP ALL DATA to export your full history and settings as a JSON file
Use RESTORE FROM BACKUP to import a previously saved backup — do this before updating versions
Export as CSV to open your readings in Excel or Google Sheets
SHARE AS IMAGE generates a visual snapshot of your current status
SECRETS
There are hidden features in this app. Find them yourself. 🐱
